## What is it?

**Voice Admin** is a .NET 10 Blazor Web app designed for Voice Coding administration. It integrates with **Talon Voice** for hands-free coding and includes a growing set of tools to make your voice workflow faster and more powerful.

---

## Key Features

- **Talon Directory Search** — Search your entire Talon voice commands directory, compile a filtered and sortable list of all your commands — perfect for when you forget a command name or need to quickly open and edit it in VS Code.
- **Custom Snippets** — Maintain a personal library of code snippets triggerable by voice. Snippets can be inserted anywhere.
- **Application Launcher** — Launch apps, folders, or websites by category or voice command or by clicking the card.
- **Hybrid Desktop Support** — A Blazor Hybrid WinForms project that lets a Windows form pop up via voice command, showing a filtered list of items to choose from — useful while coding or doing anything else hands-free.
- **Accessibility First** — Fully optimised for hands-free use; access keys throughout for keyboard/voice navigation.
