*What is it?*
VoiceLauncherBlazor is a Blazor Server app designed for voice-controlled development. It integrates with *Talon Voice* for hands-free coding and provides tools to improve your voice workflow.

*Key features:*
- Talon Directory Search — Search your Talon commands directory and produce a filtered, sortable list of commands; quickly open a command for editing in VS Code.
- Custom Snippets — Maintain a personal library of code snippets that can be triggered by voice.
- Application Launcher — Launch apps, folders, or websites by category or voice command.
- Hybrid Desktop Support — A Blazor Hybrid WinForms project that lets a Windows form pop up via voice command, showing a filtered list of items to choose from (useful while coding or working hands-free).
- Accessibility-first UI and keyboard/voice access keys.

*Quick Start (Windows)*
1. Download `VoiceLauncherBlazor-win-x64.zip` from the release page.
2. Extract and run `VoiceAdmin.exe`.
3. Full install guide: <https://github.com/Mark-Phillipson/VoiceLauncherBlazor/blob/main/docs/root-markdown/INSTALL.md|INSTALL.md>

*Run locally (dev):*
```powershell
dotnet watch --project VoiceAdmin run --urls http://localhost:5008
```
